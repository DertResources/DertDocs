Using a Template

Creating a New Repo

To use the most recent template go to this link, the default branch is stable so do not worry about getting a in development version.

Before starting this process, I’ll say that if you feel generous, please leave a star so I can see how many people have actually used the template as github does not currently have a way I can see if people are making new projects versus just visiting the site and giving up.

In the upper right hand corner click “Use this template” and then in the dropdown “Create a new repository"

On the next page leave the checkmark for “Include all branches” unchecked. Fill in your repository name and make sure the owner is correct (weather it be you or your organization) along with picking your privacy setting Click “Create repository”

Renaming your branch

In your new Repo, the only branch will be named “stable.” To fix this we can rename the branch

Click the “Branch” button second from the left

Click the 3 dots on the right hand side and pick “Rename branch”

Enter the name ‘main’ in the text box and click “Rename branch” You are now ready to get started with your new playground!
