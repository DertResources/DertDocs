Creating a Project

To get started making a project of your own design it is easiest to get started from a template or already complete project. Starting a new project on your own from scratch involves a lot of configuring build directories, adjusting settings, and getting started in simulink with game state setup. These starting points also make it easy for teaching new students, as is currently going on in DERTâ€™s programming lessons.

The template has no components and a framework for where code goes in simulink. Other than that there is nothing in the project.

The current dert project has the following components

  • CANdle
  • FMSInfo
  • HIDs
  • IMU
  • Limelight
  • ReefscapeGame
  • SwerveDrive

And the simulink modules included are

  • Swerve Drive
  • Spline Path Following
  • Odometry
  • Kalman Position Filter

Feel free to copy or learn from those examples if you need a head start, documentation on them will arrive eventually.
